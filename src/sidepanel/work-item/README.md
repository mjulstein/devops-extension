# `src/sidepanel/work-item/`

This directory contains the Active item tab UI.

## Files in this directory

- `index.ts` — barrel export for the active-item card component.
- `PinIcon.tsx` — shared visual pin icon used by the active-item cards.
- `RecentFeaturesCard.tsx` + `RecentFeaturesCard.module.css` — renders the collapsible Recent features card with persisted expand/collapse state and feature pin/set actions.
- `WorkItemCard.tsx` + `WorkItemCard.module.css` — renders child-task creation, task selection, task-state filters, and reparenting actions.
