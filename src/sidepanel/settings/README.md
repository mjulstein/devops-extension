# `src/sidepanel/settings/`

This directory contains the Settings tab UI.

## Files in this directory

- `index.ts` — barrel export for the settings tab component.
- `SettingsCard.tsx` + `SettingsCard.module.css` — renders the organization, project, assigned-to, and TODO states fields plus the Save settings and Reload extension actions.
