# Spec Skills

This directory stores optional repo-local planning skills for tools that support reusable skill or playbook loading.

Skills in this repository should stay small and focus on stable repository constraints that apply across many planning tasks.

## Files in this directory

- [`repo-planning/SKILL.md`](../../.specify/skills/repo-planning/SKILL.md) — reusable planning guidance for this repository’s Azure DevOps session model, module boundaries, browser-local storage expectations, and documentation sync rules.
