# Spec Templates

This directory stores the local markdown templates used for ideas, promoted specs, implementation plans, and task breakdowns.

These templates can be used directly or through the optional helper prompts in [`.specify/commands`](../commands/README.md).

## Files in this directory

- `idea-template.md` — template for rough ideas in `specs/ideas/`.
- `spec-template.md` — template for promoted feature specs.
- `plan-template.md` — template for implementation plans linked to a promoted spec.
- `tasks-template.md` — template for task breakdowns linked to a promoted spec.
